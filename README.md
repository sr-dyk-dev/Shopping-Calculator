# Shopping Calculator

A simple C program that calculates the total cost of items based on price and quantity.

## Description

This program prompts the user to enter the price of an item and the quantity they wish to purchase, then calculates and displays the total cost.

## Features

- Input item price (supports decimal values)
- Input item quantity (whole numbers)
- Calculates total cost automatically
- Displays a summary of the purchase

## Requirements

- GCC compiler (or any C compiler)
- Standard C library

## Compilation

To compile the program, use:

```bash
gcc shopping_calculator.c -o shopping_calc
Usage
Run the compiled program:
./shopping_calc
Then follow the prompts:
Enter the item price when prompted
Enter the quantity when prompted
The program will display the total cost
Example
Enter the Item Price: 
25.50
Enter the Item Quantity: 
3
Total Cost:
You are buying 3 items
Each costing you 25.5
And Total Cost is 76.5
